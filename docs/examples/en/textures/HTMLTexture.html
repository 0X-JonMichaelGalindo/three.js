[name]

A texture for rendering and updating HTML / CSS documents.

  • Static -- Scripts will not run, mouse and touch events will not trigger. For dynamic content, update [page:HTMLTexture.document document] style or content and call [page:HTMLTexture.redrawAsync redrawAsync].
  • Isolated (mostly) -- Stylesheets will not import, videos will not play, flash plugins will not load. <img> tags will load normally.

Code Example

const html = getMyHTMLDocumentText(); const css = getMyCSSText(); const width = 256; const height = 256; const htmlTexture = new HTMLTexture( html , css , width , height ); function render() { /* Update styles, text, etc. */ const someElement = htmlTexture.document.getElementById( 'my-element' ); someElement.classList.add( "highlighted" ); /* Redraw the texture. */ htmlTexture.redrawAsync(); renderer.render( scene, camera ); }

Constructor

[name]( [param:String html], [param:String css], [param:Number width], [param:Number height], [param:Constant mapping], [param:Constant wrapS], [param:Constant wrapT], [param:Constant magFilter], [param:Constant minFilter], [param:Constant format], [param:Constant type], [param:Number anisotropy] )

[page:String html] -- (optional) The html to draw. Defaults to a blank document.
[page:String css] -- (optional) Additional CSS to style the html.
[page:Number width] -- (optional) Texture width. Will default to 300.
[page:Number height] -- (optional) Texture height. Will default to 150.

See [page:CanvasTexture CanvasTexture] and [page:Texture Texture] for texture parameters.

Creates a new [name].

Properties

[property:HTMLDocument document]

The texture's current html document.
This will be replaced if a new html string is passed to [page:HTMLTexture.redrawAsync redrawAsync].

[property:Function onFinishRedraw]

A callback function, called when the texture has finished redrawing.

Methods

[method:Promise redrawAsync]( [param:String html], [param:String css], [param:Number width], [param:Number height] )

Parameters are the same as [page:HTMLTexture.Constructor].

Redraw any changes made to the texture's HTML document.

If html is specified, the texture's document will be completely replaced by the new html.

If width and height are specified, the texture's dimensions will change and the HTML will reflow.
